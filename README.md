# PromptDesign

## Design Intelligence for AI Agents

---

**35 Design Languages. 1 Command.**

---

### What

Structured design language specifications that enable AI agents to understand, apply, and transform UIs across any design language.

---

### Install

```bash
npm install promptdesign
```

---

### Use

**Claude / skills.sh**
```
/load promptdesign/core/minimalism
/load promptdesign/depth-texture/glassmorphism
```

**npm**
```javascript
import { getDesignLanguage } from 'promptdesign';
const design = await getDesignLanguage('minimalism');
```
