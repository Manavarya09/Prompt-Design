# Tech Futurism

## Intent
Create forward-looking interfaces that leverage cutting-edge visual treatments while maintaining usability, suggesting tomorrow's technology today.

## Principles
1. **Forward thinking** - Next-gen aesthetic
2. **Tech-forward visuals** - Advanced, sleek
3. **Clean complexity** - Sophisticated but usable
4. **Innovation showcase** - Show what's possible
5. **Functional futurism** - Beautiful AND practical

## Layout Rules
- Large, cinematic hero sections
- Asymmetric layouts
- Floating 3D elements
- Depth through layering
- Dynamic grid systems
- Generous negative space
- Edge-to-edge content
- Z-depth hierarchy

## Typography
- Font: Geometric, tech (Space Grotesk, Outfit)
- Title: 40-64px bold
- Subtitle: 20-28px medium
- Body: 16-18px regular
- Extended letter-spacing
- Uppercase for labels
- Glowing text effects
- Futuristic typefaces

## Colors
- Background: Dark (#0A0A0F) or light gray
- Primary: Electric blue (#00D4FF)
- Secondary: Purple (#8B5CF6)
- Accent: Bright cyan, magenta
- Gradients: Blue to purple
- Glow effects: Matching colors
- High contrast

## Components
- Futuristic Buttons: Glass, gradient, glow
- Holographic Cards: Floating, 3D effect
- Tech Badges: Angular, glowing
- Data Viz: 3D charts, particles
- Loading: Animated particles, DNA strands
- Floating Elements: Parallax depth
- Glass Modals: Frosted, floating
- Edge Glow: Neon border effects

## Do's
- Use dark backgrounds with light accents
- Apply subtle glow effects
- Create depth with layers
- Use geometric shapes
- Add micro-animations
- Leverage modern CSS
- Create immersive experiences
- Balance futurism with usability

## Don'ts
- Use dated design patterns
- Create cluttered interfaces
- Ignore usability
- Use excessive animation
- Apply garish effects
- Create inaccessible contrast
- Use generic layouts
- Overcomplicate interactions

## Agent Instructions
- Apply dark theme with neon accents
- Create floating, layered elements
- Use geometric shapes
- Add glow effects subtly
- Include smooth animations
- Apply futuristic typography
- Create depth through shadows
- Balance tech with usability

## Output Constraints
- HTML/CSS: Modern CSS features
- React: Animation libraries
- Glow: text-shadow, box-shadow
- Animation: CSS or Framer Motion
- Dark theme: Default
- 3D: CSS transforms or Three.js
