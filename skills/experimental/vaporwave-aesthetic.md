# Vaporwave Aesthetic

## Intent
Create nostalgic, surreal interfaces inspired by 80s/90s digital art, featuring pastel colors, Greek statues, and glitch effects.

## Principles
1. **Retro-futurism** - Past meets digital future
2. **Nostalgic dreamscape** - Liminal space feeling
3. **Maximum aesthetic** - Everything is styled
4. **Pastel neon** - Soft but bright
5. **Cultural sampling** - Classical meets digital

## Layout Rules
- Full-bleed gradient backgrounds
- Grid patterns (Greek temples, checkerboards)
- Centered, balanced compositions
- Floating elements
- 80s-inspired window chrome
- Oversized elements
- Layered depth
- Perspective grids

## Typography
- Font: Wide, italic (Arial Black Italic, Vaporwave fonts)
- Double-line text effects
- Pastel colors
- Title: 36-72px
- Body: 16-18px
- Letter-spacing: 2-5px
- Slight rotation effects
- Chrome-style text

## Colors
- Background: Dark purple (#1A0A2E) or pink (#2D1B4E)
- Pastel pink: #FF71CE
- Pastel cyan: #01CDFE
- Pastel blue: #05FFA1
- Pastel purple: #B967FF
- Yellow: #FFFB96
- Grid colors: Pink/blue gradients
- Checkerboard: Black and pink

## Components
- Vapor Buttons: Gradient fill, chrome edge
- Sun Elements: Retro sun graphics
- Grid Backgrounds: Perspective grids
- Greek Elements: Statue imagery
- Chrome Text: Metallic text effect
- Checkerboard: Black/pink patterns
- Floating Windows: 80s window style
- Holographic Effects: Rainbow shifts

## Do's
- Use pink/cyan/purple palette
- Add retro sun graphics
- Include perspective grids
- Create chrome text effects
- Apply checkerboard patterns
- Use wide, italic fonts
- Add floating elements
- Create dreamlike spaces

## Don'ts
- Use realistic colors
- Create minimal designs
- Apply modern aesthetics
- Ignore the nostalgia
- Make it subtle
- Use flat backgrounds
- Follow current trends
- Create professional interfaces
