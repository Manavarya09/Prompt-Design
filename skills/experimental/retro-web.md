# Retro Web (90s HTML Style)

## Intent
Celebrate the nostalgic aesthetics of early web design, recreating the charm of 90s internet with its characteristic boldness and simplicity.

## Principles
1. **Nostalgic authenticity** - True to the era
2. **Bold simplicity** - Basic but memorable
3. **Web 1.0 charm** - Before web standards
4. **Functional beauty** - Form follows function
5. **Tangible digital** - Retro-futuristic feel

## Layout Rules
- Fixed width centered container (800px max)
- Table-based layouts (real or simulated)
- Banner headers at top
- Navigation bars
- Multiple columns
- Footer at bottom
- Visible page boundaries
- Beveled borders

## Typography
- Font: Times New Roman, Arial, Comic Sans
- Times for body text
- Bold headlines
- Sizes: 12pt body, 18pt+ headlines
- Blue links (default browser blue)
- Visited links purple
- Underlined text
- No web fonts

## Colors
- Background: Gray (#C0C0C0), white, or navy
- Links: Blue (#0000EE), visited (#551A8B)
- Text: Black
- Tables: White with gray borders
- Beveled buttons: 3D effect
- Marquee colors: Multiple
- Animated GIF colors: 256 max
- Gradient backgrounds: Fixed-position bars

## Components
- Retro Buttons: 3D beveled, click animation
- Table Layout: Bordered cells
- Hit Counters: Classic visitor counter
- Guestbook Links: Form links
- Marquee Text: Scrolling announcements
- Tiled Backgrounds: Repeating patterns
- Under Construction Signs: GIF placeholders
- Web Rings Navigation: Previous/Next links

## Do's
- Use classic web 1.0 colors
- Apply beveled 3D effects
- Include table layouts
- Use system fonts
- Add blinking text
- Create framed sections
- Use thick borders
- Include nostalgic elements

## Don'ts
- Use modern CSS
- Apply flat design
- Use web fonts
- Create responsive layouts
- Minimize anything
- Add modern touches
- Use gradients for depth
- Follow current patterns

## Agent Instructions
- Use fixed-width layouts (800px)
- Apply 3D beveled buttons
- Use gray backgrounds
- Set blue/purple links
- Include table structures
- Use Times New Roman
- Add thick borders
- Create classic web 1.0 look

## Output Constraints
- HTML: Table layouts (real or styled)
- CSS: Simulate retro effects
- Width: Fixed 800px
- Font: Times, Arial, system
- Colors: Classic web palette
- Borders: 2-3px beveled
