# Cyberpunk UI

## Intent
Create futuristic interfaces inspired by dystopian sci-fi, featuring neon colors, tech aesthetics, and digital decay.

## Principles
1. **High-tech feel** - Advanced but edgy
2. **Neon dominance** - Glowing lights everywhere
3. **Digital grit** - Tech wear and tear
4. **Dystopian mood** - Cool but slightly ominous
5. **Futuristic contrast** - Light against dark

## Layout Rules
- Dark backgrounds (#0D0D0D to #1A1A2E)
- Neon accent edges
- Scanline overlays
- Glitch-capable elements
- HUD-style frames
- Asymmetric layouts
- Data visualization prominent
- Terminal-style sections

## Typography
- Font: Monospace, futuristic (Share Tech Mono, Orbitron)
- Glowing text effects
- Title: 28-40px bold, neon glow
- Body: 14-16px monospace
- Terminal-style content
- Letter-spacing: 1-3px
- Uppercase for labels
- Blinking cursor effects

## Colors
- Background: Deep black (#0D0D0D) or dark blue
- Neon pink: #FF0080
- Neon cyan: #00FFFF
- Neon yellow: #FFE600
- Neon green: #39FF14
- White text: #FFFFFF
- Glitch reds: #FF0000
- Purple accent: #9D00FF

## Components
- Cyber Buttons: Neon border, glow on hover
- HUD Frames: Angular, tech-style borders
- Terminal Windows: Dark with green text
- Data Streams: Animated data visualizations
- Glitch Text: Randomized distortion
- Scanline Overlays: CRT effect
- Holographic Cards: Glass with neon edges
- Tech Badges: Angular, glowing

## Do's
- Use neon colors on dark backgrounds
- Add glow effects to text
- Include scanlines
- Create glitch effects
- Apply angular shapes
- Use monospace fonts
- Add digital artifacts
- Make it feel futuristic

## Don'ts
- Use light backgrounds
- Apply soft, rounded design
- Create friendly interfaces
- Use pastel colors
- Ignore the mood
- Add unnecessary warmth
- Follow conventional patterns
- Make it too readable

## Agent Instructions
- Apply dark backgrounds
- Use neon colors (#FF0080, #00FFFF, #FFE600)
- Add glow: text-shadow with color
- Include scanline overlay
- Use monospace fonts
- Create angular shapes
- Add glitch effects
- Apply CRT-style elements

## Output Constraints
- HTML/CSS: Glow effects, animations
- React: Glitch component, neon effects
- Glow: Multiple text-shadows
- Animation: Glitch keyframes
- Colors: Neon on dark
- Scanlines: CSS overlay
