# Pastel UI

## Intent
Create soft, approachable, and friendly interfaces using muted pastel colors that reduce visual fatigue while maintaining pleasant aesthetics.

## Principles
1. **Muted softness** - No harsh, saturated colors
2. **Friendly aesthetic** - Welcoming and approachable feel
3. **Reduced eye strain** - Lower visual intensity
4. **Balanced pastels** - Harmonious color relationships
5. **Youthful but professional** - Suitable for various audiences

## Layout Rules
- Light backgrounds (cream, white, or light pastel)
- Generous whitespace
- Rounded corners: 12-20px
- Soft card elevations
- 20-24px spacing between elements
- Floating elements with gentle shadows
- Organic, friendly shapes
- No harsh edges anywhere

## Typography
- Font: Rounded sans-serif (Nunito, Quicksand, Poppins)
- Soft weight hierarchy
- Title: 22-28px semibold
- Body: 14-16px regular
- Line-height: 1.5-1.6
- Dark gray text for readability
- Subtle text shadows optional
- Friendly, approachable letterforms

## Colors
- Background: White, cream (#FFFBF0), or very light pastel
- Pastel pink: #FFD1DC, #FFB6C1
- Pastel blue: #AEC6CF, #B5EAD7
- Pastel purple: #C9B1FF, #E0BBE4
- Pastel yellow: #FFF5BA, #FEF3C7
- Pastel green: #C1E7E3, #B5EAD7
- Pastel mint: #B8E0D2, #D6FFFA
- Text: Dark gray (#4A5568)
- Subtle borders: Pastel tinted

## Components
- Pastel Buttons: Soft fill, rounded pill
- Pastel Cards: Light pastel backgrounds
- Pastel Badges: Small tinted pills
- Pastel Inputs: Light border, soft focus
- Pastel Progress: Soft gradient fill
- Pastel Tags: Tinted rounded labels
- Pastel Avatars: Soft colored circles
- Pastel Dividers: Subtle gradient lines

## Do's
- Use muted, desaturated pastels
- Keep background light
- Apply generous rounded corners
- Create soft shadow effects
- Maintain readable text contrast
- Use harmonious color combinations
- Add subtle hover states
- Keep the interface soft overall

## Don'ts
- Use saturated, bright colors
- Create harsh contrasts
- Apply dark backgrounds
- Use bold, aggressive typography
- Add sharp corners
- Create cluttered interfaces
- Use clashing pastel combinations
- Apply heavy shadows

## Agent Instructions
- Select muted pastel palette
- Use light backgrounds
- Apply large border-radius (12-20px)
- Create soft shadows
- Ensure 4.5:1 text contrast
- Use rounded fonts
- Maintain gentle spacing
- Keep all elements soft

## Output Constraints
- HTML/CSS: CSS custom properties for pastels
- React: Pastel theme component
- Saturation: 20-40% max
- Border-radius: 12-20px
- Contrast: 4.5:1 minimum
- Shadow: Soft, low opacity
- Background: Light only
