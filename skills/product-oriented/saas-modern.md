# SaaS Modern (Stripe-style Clean UI)

## Intent
Create professional, trustworthy interfaces for software products that convey reliability, encourage conversion, and reduce friction.

## Principles
1. **Trustworthy appearance** - Professional without being corporate
2. **Conversion-focused** - Every element supports action
3. **Minimal cognitive load** - Simple, obvious paths
4. **Premium feel** - Quality without stuffiness
5. **Progressive disclosure** - Show what's needed, hide complexity

## Layout Rules
- Max-width: 1200px centered
- 8px base spacing unit
- Generous whitespace
- Sidebar navigation (collapsible)
- Card-based content organization
- Hero section with value prop
- Pricing tables prominent
- Clear visual hierarchy

## Typography
- Font: Inter, Plus Jakarta Sans, or DM Sans
- Title: 32-48px semibold
- Subtitle: 18-24px regular
- Body: 14-16px regular
- Small: 12-13px for labels
- Weight: Light (300) to Bold (700)
- Line-height: 1.5-1.6

## Colors
- Primary: Blue (#0A2540) or indigo
- Accent: Vibrant blue (#635BFF) or purple
- Background: White or very light gray
- Text: Dark slate (#1A1A2E)
- Secondary: Medium gray (#6B7280)
- Success: Green (#10B981)
- Borders: Light (#E5E7EB)
- Subtle gradients

## Components
- Clean Buttons: Solid or ghost, 8px radius
- Cards: White, subtle border, hover lift
- Forms: Clean inputs with floating labels
- Tables: Clean with hover rows
- Modals: Centered, dark overlay
- Sidebar: Icon + text nav
- Empty States: Friendly illustrations
- Onboarding: Step indicators

## Do's
- Use generous whitespace
- Create clear CTAs
- Show pricing clearly
- Include trust signals
- Support smooth onboarding
- Add helpful tooltips
- Show loading states
- Provide clear feedback

## Don'ts
- Use cluttered layouts
- Create confusing navigation
- Hide pricing
- Overwhelm with features
- Use corporate colors
- Add unnecessary friction
- Use low-contrast text
- Create long forms

## Agent Instructions
- Apply clean, spacious layout
- Use professional color palette
- Create obvious CTAs
- Show pricing prominently
- Include trust signals
- Support step-by-step flows
- Add clear form validation
- Ensure professional appearance

## Output Constraints
- HTML/CSS: Tailwind-style utility classes
- React: Component library pattern
- Max-width: 1200px
- Spacing: 8px base
- Border-radius: 8px
- Professional appearance
- Conversion-focused
